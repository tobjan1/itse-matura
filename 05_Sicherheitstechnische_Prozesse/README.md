# 05 Sicherheitstechnische Prozesse

Wie IT-Sicherheit als gelebter, messbarer und kontinuierlich verbesserter Prozess organisiert wird. Dieses Thema umfasst die Grundlagen des Prozessmanagements, das RACI-Modell für Verantwortlichkeiten, den Deming-Cycle (PDCA) als zentrales Werkzeug der ISO 27001 und das Reifegradmodell zur Prozessbewertung.

## Prüfungsrelevante Kernthemen

- **RACI**: Alle vier Rollen erklären, muss/kann-Regel beherrschen
- **PDCA**: Alle vier Phasen + ISO 27001-Abschnitte + die drei Check-Aspekte (Konformität / Effektivität / Effizienz)
- **Reifegradmodell**: Alle sechs Stufen (0–5) mit Schlüsselmerkmal nennen können
- **SMART**: Akronym ausschreiben, Beispiel nennen
- **Prozessdokumentation**: 2-stufige Freigabe erklären
